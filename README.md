# SecureBank Portal
A simple banking web application built using Java Enterprise technologies. This project was developed as a Semester 6 project for the subject **Java Enterprise Technology**.

## Contributors
- Ayush Warrier
- Rajat Gupta

## Tech Stack
- Java Servlets
- JSP (Java Server Pages)
- HTML, CSS
- MySQL
- JDBC

## Features
- User login
- View account balance
- Transfer money
- View transaction history

## NOTE
This is an academic project and does not include advanced security features.
